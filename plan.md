## M2. 2콜 분리 (판정콜=코드 → 대사콜=LLM)

- ⬜ LLM 프록시(Next API 라우트)로 API 키 서버측 보관
- ⬜ 대사콜: "이미 정해진 판정 결과"를 전원 담당자 인물 대사로 연기 (수용/거절을 바꾸지 못함)

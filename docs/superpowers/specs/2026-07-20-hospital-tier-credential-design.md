# 설계 스펙 — 병원 등급 = 획득한 법적 자격 (개원 위저드 파생 라벨)

## 1. 목적 & 스코프

**목적**: 개원 위저드에서 플레이어가 필수 배후과를 채우면, 그 채용에서 **법적 응급의료기관 등급이 파생**돼 실시간 라벨로 뜬다. 등급은 "고르는 슬라이더"가 아니라 **"요건을 채워 획득하는 자격"**이다 — 상위 등급일수록 적자 필수과를 더 떠안아야 얻으므로, **정체성과 딜레마가 한 라벨에** 실린다. 이름 없는 병원이 "○○병원 · 지역응급의료센터"라는 격을 가진 기관이 된다.

**핵심 원칙(A안)**: 등급은 **판정·경제 불변식을 한 줄도 안 건드린다.** `backupCare`를 읽는 순수 파생 함수 1개 + 위저드 라벨이 전부다. `FIXED_BEDS`·예산·`adjudicate`·불변식 I8 모두 불변.

**스코프**
- **IN**: `hospitalTier()` 파생 함수 + `HospitalTier` 타입 + 위저드 실시간 등급 라벨(+ 4단 등급 사다리에 현위치).
- **OUT(확장점 §8)**: 등급이 실제 용량 변경(B/C안) · 재지정/철회(연 단위·다주차) · 지역 신문 결과층 · 결말/장부 등급 표시 · 고유명에 등급 명칭 입력 차단 · 24시간 가중 임계(권역 예산벽).

## 2. 왜 A안이 이 게임에 맞나 (근거 요약)

- **법 그대로**: 등급 = 지정 명칭(자칭 불가), 요건 충족으로 획득. [emergency-tier-designation-law.md](../../research/emergency-tier-designation-law.md).
- **안티-퍼즐**: 이익 최대화(미용 10명 = 미지정, +700/주)와 등급 상승(필수과 적자)이 **정반대로 당긴다** → 지배 전략 없음. 등급 사다리가 "이익을 위해 포기한 격"을 legible하게 만든다(포기도, 감수도 둘 다 진다).
- **show-don't-tell**: "자격 미달/더 뽑으세요" 카피 없이 **상위 명칭이 안 열리는 것으로** 보여준다. 등급 라벨은 중립 사실(현재 어느 지정에 해당하나)이지 이익·소송 해석이 아니다.
- **불변식 0 침습**: 마감(2026-08-10) 대비 최저 비용 — 파생 함수 1개 + UI 라벨.
- **plan F4 정합**: F4가 이미 "콜이 아니라 병원 등급 · 위저드/결말 자리 · 하드 게이트가 아니라 압력 · 지정은 연 단위"로 규정했다. A안은 파생 라벨이라 정확히 **압력·정체성**이고 within-week 게이트가 아니다(연 단위 척도 존중 — 재지정은 다주차 확장점).

## 3. 등급 사다리 (법 근거, 임계는 각색)

플레이어 관련 등급만(중앙 = 전국 1개 · 전문 = 소아/화상/독극물은 범위 밖):

| 필수 배후과 수 (`backupCare.length`) | 등급 |
|---|---|
| 0 | 미지정 (일반 병원) |
| 1 | 지역응급의료기관 |
| 2 | 지역응급의료센터 |
| ≥3 | 권역응급의료센터 |

- **서열(더 많은 배후 = 상위)만 근거**, 임계 수치는 각색이다 — 실제 별표는 전문의·간호사·병상 수이나, 게임의 기존 축인 **'배후과 수'로 추상화**한다([starting-world §2](../../research/current-korea-starting-world.md)).
- **예산 상호작용(현 카탈로그)**: 필수과 1명씩 최저 조합으로 지역기관(외과 22) → 지역센터(+산부 28 = 50) → 권역(+순환기 30 = 80)까지 도달 가능, 남는 예산으로 미용. 즉 **사다리 전체가 도달 가능**하되 **오를수록 미용(이익) 자리가 준다 = 딜레마**.
- ⚠️ **4개 필수 배후과(순환기·산부·신경·외과 = 콜 4종)를 다 갖추려면 110억 > 100억**이라 **어떤 등급도 4종 응급을 다 못 받는다**(기존 설계 "한 병원이 4과 다 못 갖춘다" 보존). **권역이어도 최소 1종은 벽** — 등급이 정체성을 줘도 아무도 다 구하지 못한다는 논지가 유지된다.

## 4. 정의 (코드)

```ts
// types.ts
export type HospitalTier = 'UNDESIGNATED' | 'LOCAL_INSTITUTION' | 'LOCAL_CENTER' | 'REGIONAL_CENTER'

// tier.ts (신규)
export const TIER_LABELS: Record<HospitalTier, string> = {
  UNDESIGNATED: '미지정 (일반 병원)',
  LOCAL_INSTITUTION: '지역응급의료기관',
  LOCAL_CENTER: '지역응급의료센터',
  REGIONAL_CENTER: '권역응급의료센터',
}

/** 필수 배후과 수 → 법적 등급. backupCare만 읽는 순수 파생(미용·검진은 등급과 무관). */
export function hospitalTier(backupCareCount: number): HospitalTier {
  if (backupCareCount >= 3) return 'REGIONAL_CENTER'
  if (backupCareCount === 2) return 'LOCAL_CENTER'
  if (backupCareCount === 1) return 'LOCAL_INSTITUTION'
  return 'UNDESIGNATED'
}
```

- 위저드는 현재 `choices`에서 **필수 배후과 수**(= `providesBackup` 있고 `count>0`인 과 수)를 세어 `hospitalTier()`에 넘긴다 — `buildHospital`의 `backupCare` 파생과 **같은 로직**이라 단일 출처(드리프트 방지).
- **`Hospital`에 `tier` 필드를 저장하지 않는다** — `backupCare`에서 언제든 파생한다(저장은 드리프트 초대, [자동 인덱스·PR번호 생략]과 같은 발상: 파생 가능한 값은 이중 기재하지 않는다). UI·결말이 필요하면 그 자리에서 `hospitalTier(hospital.backupCare.length)`.

## 6. 톤·show-don't-tell 가드레일

- 등급 라벨 = **중립 사실**. "자격 미달 · 더 뽑으세요" 류 유도·해석 카피 금지 — 상위 명칭이 안 열리는 것으로 보여준다.
- 사다리에 **요건(필수 배후과 수)은 표시 가능**(공개된 법 요건이다) — 단 **이익·소송은 여전히 숨긴다**(기존 위저드 원칙). 등급↑의 비용(적자)은 플레이어가 예산 잔액·장부에서 스스로 발견한다.
- **실명·정파 없음**: 등급은 응급의료법 지정 체계라 특정 인물·집단 지목이 아니다. 안전.

---

## 7. 알아둘 천장 (정직하게)

- MVP의 등급은 **정체성·압력 레이어**지 새 판정 게이트가 아니다(A안의 본질). 판정은 여전히 `backupCare`·자리가 가른다. "등급이 게임을 실제로 바꾼다"는 B/C안이나 재지정(다주차)에서 온다.
- 임계(0/1/2/3)는 각색·튜닝값이다. **사다리가 딜레마를 만드는지(미지정↔권역 스펙트럼)는 브라우저 7일 완주로만** 확인된다 — 불변식 I8은 안 건드리나 "체감 밸런스"는 테스트로 안 잡힌다.
- **4종 콜 vs 도달가능 3배후과의 부등호**가 유지돼야 "권역이어도 벽"이 성립한다 — 카탈로그 `hireCost` 튜닝 시 재확인(현재 4배후과 = 110억 > 100억).

## 8. 확장점 (다음 슬라이스가 여기에 붙는다)

- **24시간 가중 임계**: 권역 = 3배후과 **전부 24h(roundTheClock)** → 6명 필요·160억 > 100억이라 **예산벽(도달 불가)**. "최상위 격은 애초에 못 산다"는 강한 메시지지만 권역이 유령 등급이 됨 → 밸런싱 결정으로 보류.
- **등급을 결말·주간결산에 병원 정체로 표시** (파생 읽기라 저비용).
- **재지정/철회(다주차·연 단위)**: 등급을 유지 못 하면 강등 — 매주 재구성 축(world 이벤트)과 결합(본선). plan F4의 "지정·평가·철회 연 단위" 부분.
- **고유명에 등급 명칭 입력 차단/경고**("'권역응급의료센터'는 지정 명칭 — 요건을 갖춰 획득하세요") = 방금 리서치한 **제59조를 인터랙션으로**. 단 살짝 'telling'이라 MVP 밖.
- **지역 신문 결과층**(별도 슬라이스): 지방일수록 거절의 대가↑.
- **B/C안 승격**(등급이 실제 병상·예산 변경): 게임 무게↑ 필요 시. **기각 사유(현재)**: B/C는 자리<콜·I8 불변식과 밸런싱을 건드려 마감 리스크↑ + 난이도 슬라이더 오독 위험. A안이 정체성·안티-퍼즐·저비용을 다 충족해 MVP엔 불필요(YAGNI).

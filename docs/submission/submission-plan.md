## 🔑 선결 — 오늘(7/16) **사용자가** 확인/결정할 3가지

코드가 아니라 규정·계정 문제라 내가 대신 못 한다. 이게 막히면 뒤 일정이 통째로 흔들리니 **가장 먼저**.

1. **제출 요강/폼 전문 정독 → 필수항목·마감 시각 확정.** 특히
   - (a) 빌드 제출 형식: "라이브 Vercel URL만" 허용인가, 아니면 **다운로드 가능한 웹빌드 아티팩트(zip 등)도 별도 요구**인가?
   - (b) **마감 정확 시각·타임존** (8/10 00:00 vs 23:59 KST?).
   - (c) PDF 규격/용량(폼상 1GB 확인됨).
   → 폼을 네가 갖고 있으니 같이 확정하자.
2. **Anthropic 콘솔에서 결제/크레딧 활성 상태 직접 확인.** 결제 설정은 내가 대행 불가(금지 영역) → 네가 직접. 미설정이면 실LLM(X8)은 **처음부터 스킵 전제**로 잡고 ④PDF를 "설계 + 로컬 시연" 중심으로 쓴다.
3. **대회 FAQ '유료 API 비용 부담 주체' 문의 발송 여부 결정.** 초안은 내가 준비하되, **발송은 네 승인 후**.

## 최소 제출 가능선 (min cut line) — 목표 **8/1 (U5)**

LLM/유료 API가 **전혀 없어도**(결정론 폴백 대사) Vercel 공개 URL에서 **한 주가 마우스만으로 처음부터 끝까지 완주**되고(개원 → 7일 진료 → 주간 결산 → 결말), 그 한 판이 데모 핵심을 화면에서 모두 보여주는 상태:

1. 응급 콜 도착 — 분만·뇌출혈·중증외상·심근경색이 예고 없이 섞여 온다(플레이어=받는 벽)
2. 콜마다 수용/거절 → `NO_BED`/`NO_BACKUP_CARE` 등 하드락 사유 태그 누적, 못 받은 응급은 이튿날 아침 신문에
3. 발신자가 아무리 매달려도 **배후진료(그 과) 없으면 어떤 문장으로도 안 받아짐**(하드락 벽 가시화)
4. 결말 — 누적 병원 장부(흑자↔필수과 채용 0) + 그 주 신문(못 받은 응급 종류별 헤드라인) 병치(해석 카피 0) + README 배포 링크·실행법

이 선이 서면 ①빌드+소스가 완성되고, 이 화면으로 ②영상·③④PDF 집필이 모두 가능해져 **4종 전부 제출 가능**.

## LLM 삽입 지점 (판정=코드 / 대사=LLM)

최소선(U5) 확보 후 `X8`에서 얹는다. 판정은 코드가 이미 확정, **LLM은 그 결과를 인물 대사로 연기만** → 게임 상태 전이는 LLM 손에 안 넘어감(정합성 테스트로 타입·런타임 양쪽 잠금). 무키·API 실패·상한 초과 시 결정론 폴백으로 **자동 강등**. 기존 폴백 경로는 삭제하지 않고 그 위에 옵셔널 레이어로만 끼운다 → LLM은 전량 upside, 실패해도 되돌릴 최소선이 항상 손에 있다. **이 강등 구조 자체가 ④AI PDF의 셀링**(설득으로 병상0 벽을 못 뚫는 붕괴를 타입 수준에서 차단)이 된다.

## 플레이 영상 샷 구성 (30~60초)

U5 시드 고정 배포 화면 실플레이. 목표: **'받는 벽의 무력감'과 '흑자 병원이 왜 그 벽이 되었나'**를 60초 감정 곡선으로 각인.

| 구간 | 내용 |
|---|---|
| 0–5초 | 콜드오픈 — 개원 선택(미용·검진 vs 필수과) + 첫 응급 콜 도착 카드(자막: 응급 종류·요구 배후과) |
| 5–20초 | 코어 임팩트 — 분만·뇌출혈·중증외상 콜이 연달아, 배후진료 없어 `NO_BACKUP_CARE` 하드락 누적(받고 싶어도 못 받음). **여기가 라이브 AI 대사 구간** |
| 20–30초 | 절정 — 발신자가 아무리 매달려도 '그 과가 없으면' 안 받아짐(구조의 벽을 받는 쪽에서) |
| 30–40초 | 이튿날 아침 신문 — 어제 못 받은 응급들이 종류별 헤드라인으로(내 결정의 흔적이 세계에 남는다) |
| 40–55초 | 결말 병치 — 누적 장부(미용·검진 흑자 ↔ 필수과 채용 0) + 그 주 신문. 해석 카피 없이 두 숫자 면만 정중앙 |
| 55–60초 | 태그라인 "판정=결정론 코드 / 대사=LLM — 사정이 딱해도 배후진료 없이는 못 받는다" + 타이틀 |

결정론 폴백으로도 100% 촬영 가능하게 각본을 짜되, U2 극소판으로 5–20초 대사 구간은 실제 AI 대사를 담는다.

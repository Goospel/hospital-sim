## 진행 상태

> 이 체크리스트는 `npm run pdf`가 읽어 **미완료 항목을 매번 콘솔에 찍는다**([scripts/build-submission-pdfs.mjs](../../scripts/build-submission-pdfs.mjs)). 제출 문서를 건드릴 때마다 강제로 눈에 들어오게 하려는 것 — 규약을 지키는 것과 규약이 읽히는 것은 다르다.
>
> **형식을 바꾸지 말 것**: 스크립트는 `- [ ]` / `- [x]` 로 시작하는 줄만 센다.

- [x] **①** 플레이 가능한 빌드 + 전체 소스 — **GitHub Pages 라이브**(요강 비고와 동일) + Vercel 라이브 + GitHub 공개 저장소 *(저장소 URL 확정치 기입 필요)*

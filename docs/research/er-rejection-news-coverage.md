## 0. 🔴 먼저 — 윤리 가드 (읽고 시작할 것)

대구 추락 10대(2023-03), 용인 70대(2023-05), 보은 33개월(2024-03), 부산 고교생(2025-10)은 **유족이 있는 실제 사건**이다. 허구 헤드라인이 이걸 연상시키면 **2차 가해**다.

**게임 헤드라인이 지켜야 할 규칙**:

1. **지역·연도·환자 프로필 중 최소 두 개를 실제와 어긋나게** 고정한다.
2. **실명·실제 병원명 금지.** 실제 보도조차 병원명을 대부분 안 밝힌다 — **익명이 오히려 더 사실적이다.** "○○대병원" 같은 가명도 실존 병원을 연상시키면 위험.
3. **지역명은 가공**으로 통일한다.
4. `game-concept.md` §6과 일관: 자극적 죽음 묘사 금지, 특정 집단 비난 금지.

> ⚠️ 이 규칙은 **헤드라인 뱅크 파일 상단 주석에 그대로 옮겨 적는다.** 나중에 헤드라인을 추가하는 사람이 이 문서를 안 읽어도 지키게.

## 1. 헤드라인의 문법

### 지배 골격

```
[지역/환자] '뺑뺑이' + [N]곳 거부 … [T]시간 만에/끝에 + [사망 동사]
```

**말줄임표(…)가 절 구분자다.** 표본 대다수가 마침표도 쉼표도 아닌 `…`로 **원인절과 결과절을 잇는다**. `A…B` = "이런 일이 있었고 → 그래서 죽었다". 이게 사실상의 문법이다.

### 슬롯별 실제 용례

| 슬롯 | 실제 표현 | 규칙 |
|---|---|---|
| **거부 수량** | "15곳서 거부", "병원 12곳 거절", "14번 거절당해", "병원 14곳 돌다" | **숫자가 앞부분에 온다.** 숫자 = 충격의 단위 |
| **시간** | "5시간 뺑뺑이 끝에", "2시간 헤매다", "1시간 넘게", "3시간 반만에" | "끝에" / "만에" / "넘게" |
| **이동 동사** | **돌다 / 헤매다 / 전전하다 / 찾다가** | "돌다"가 압도적('뺑뺑이'와 어울림) |
| **사망 동사** | **사망 / 숨졌다 / 숨져 / 끝내 사망 / 결국 사망** | 🔴 **"죽었다"는 절대 안 씀.** "숨지다"가 기사체 |
| **필연 부사** | "결국", "끝내", "끝에" | 죽음을 **예정된 귀결**로 배치 |
| **반복 표지** | ⭐ **"또"** | 개별 사건이 아니라 **패턴의 n번째**임을 선언 |
| **환자 지칭** | "70대", "10대", "고교생", "4살 남아", "33개월 아이", "태아", "산모" | **이름 없음. 나이+속성만.** 익명화가 규칙 |
| **태그** | `[속보]` `[단독]` `[전국]` | 대괄호 |

### 인용부호 규칙

🔴 **'뺑뺑이'는 거의 항상 작은따옴표 안에 있다.** 속어라 언론이 거리를 두는 장치. **이게 빠지면 기사가 아니라 게임 텍스트로 읽힌다.**

변주: `'응급실 뺑뺑이'`(기본) / `'구급차 뺑뺑이'` · `'병원 뺑뺑이'` · `'분만실 뺑뺑이'`(상황별 교체) / `'응급실 뺑뺑이 사망'`(사건명화) / 따옴표 없음(용어 정착 후 통계 기사)

### 표기 함정

- **말줄임표는 세 점 문자 하나(U+2026)**. `...`(마침표 3개)로 쓰면 한국 기사처럼 안 보인다. 단 YTN처럼 `...`를 쓰는 매체도 실재하니 **매체별 변주로는 활용 가능**.

## 2. ⭐ 프레이밍은 시간에 따라 이동한다

**가장 중요한 발견.** 게임의 후속 헤드라인 설계에 직접 쓴다.

### 1단계 — 발생 직후: **무주체(無主體)**

가해자를 지목하지 않는다. "병원 15곳서 거부"는 **어느 병원인지 안 밝힌다.** 의사 이름도 없다. 환자도 익명. **아무도 이름이 없는 사건**으로 보도된다.

> 📌 **이건 우연히도 이 게임의 show-don't-tell 원칙과 정확히 같다.** 뉴스가 "당신 탓"이라고 말할 필요가 없고, **실제 언론도 안 그런다.** 숫자(15곳, 5시간)가 스스로 말하게 둔다.

### 2단계 — 통계·분석: **구조**

지목 대상: 전문의 부족, 전공의 공백, 중환자실 병상, 지방 인력난, 권역 간 자원 불균형, **배후진료 부재**.

### 3단계 — 사법 국면: **개인 대 구조의 정면 충돌**

| 개인 책임 쪽 | 구조 책임 쪽 |
|---|---|
| "치료 거부한 의사 2명"(경향) | "뺑뺑이 문제를 의료진 개인에게 전가"(의료계 인용) |
| "진료 기피 의사에 벌금형"(로앤비) | "가혹하고 부당"(시사저널, 의료계 인용) |
| "환자 거부 병원 보조금 중단 정당"(한경) | "응급환자 수용은 의사 개인의 의지가 아니라 수술실·중환자실 여력, 배후 진료 인력이 맞아야 가능" |

### 경향성

- **발생 기사는 거의 100% 구조·무주체 프레임** — 개인을 탓하지 않는다
- **개인 책임 프레임은 사법 절차가 있을 때만 등장**하고, 등장하는 즉시 **반론이 같은 줄에 붙는다**
- 언론 자신이 "누가 잘못했다"고 단정하는 헤드라인은 **거의 없다**

⚠️ 이 비율은 **수집한 38건 표본 기준**이며 '뺑뺑이' 키워드에 편향돼 있다. **정량적 프레임 비율 주장은 미확인.**

### 📌 게임 적용

실제 보도는 사망 기사로 끝나지 않고 **몇 년 뒤 송치/판결 기사**가 붙는다(대구 건: 사건 3년 만인 2026-06-16 의사 2명 불구속 송치). **게임 결말에서 "3년 만에 의사 2명 송치" 같은 후속 헤드라인이 뜨고, 그 의사가 플레이어라면** — 실제 보도 구조를 그대로 재현하는 것이 된다.

## 3. 거부 사유는 어떻게 보도되나

**패턴: `[병원 사유 직접인용] + [반박 사실 병치]`**

| 실제 보도 표현 | 맥락 |
|---|---|
| "병상이 부족하다" / "병실 부족 등의 이유로" | 가장 흔함. 중립 전달 |
| "신경외과 전문의가 없다" / "전문의 부재" | 구조 원인으로 연결 |
| "소아 진료 불가" / "소아 심정지 불가" | **심정지 후에도 거부**한 사실을 병치해 사유의 부당성을 드러냄 |
| "정신건강의학과 진료 등이 필요해 보인다" | 외상 환자를 정신과로 미룸 → "중증도 분류를 하지 않은 채"와 나란히 배치 |
| "인력·병상 부족" | 통계 기사의 관용구 |

**기법 예시**: "응급실 병상 비었는데 '뺑뺑이' 사망"(KBS) — 사유와 사실을 헤드라인에서 정면 충돌시킨다. "환자 집에서 '5분 거리'에 있는 종합병원 등 15개 병원이 수용을 거부" — 거리를 대비시킨다.

> 📌 게임의 `RejectionReason` 5종 중 **4종**(NO_BED / NO_ER_ONCALL / ER_OVERCROWDED / NO_BACKUP_CARE)이 위 보도 표현과 그대로 대응한다. 나머지 **NO_NIGHT_BACKUP**(배후과는 있으나 야간 당직 공백)은 개별 사건 보도의 사유 표현이 아니라 **§4의 수용곤란 고지 사유 중 증가율이 가장 큰 '인력부족'** 쪽에 대응한다 — 과 자체가 없는 NO_BACKUP_CARE와는 **다른 사유**다. ## 4. 통계

### 응급실 수용곤란 고지 (소방청 → 서영석 의원실)

| 연도 | 건수 |
|---|---|
| 2023 | **58,520** |
| 2024 | **110,033** (약 2배) |
| 2025 (8월까지) | 83,181 |

**2024년 사유별**: **인력부족 18,750 → 43,658 (2.3배)** / 기타 26,583 → 52,050 / 장비부족 1,500 → 1,999 / 병실부족 9,680 → 12,041

### 119 재이송 (환자를 태운 채 병원을 못 찾아 다시 이송)

2023년 **4,227건** → 2024년 **5,657건**(+34%). 사유: **전문의 부재 2,331건**(2023년 1,771) / 병상 부족 720건(2023년 635)

> ⚠️ **수용곤란 고지(11만)와 재이송(5,657)은 다른 지표다.** 전자는 병원이 미리 "못 받는다"고 고지한 건수, 후자는 실제로 환자를 태우고 다시 병원을 찾은 건수. **헤드라인에서 혼용되니 게임에 쓸 때 구분할 것.**

### 이송 시간 악화 (소방청)

30분 초과 이송 비율: 2023년 **1.9%** → 2024년 **3.8%** → 2025년 상반기 **5.4%**. 현장도착→병원도착 **120분 초과** 사례 2024년에 전년 대비 **80.3% 급증**. 2023년 전체 이송 모수 179만 건.

### 경향신문 보도사례 분석 (2024-02-20 ~ 08-24, 보도된 34건)

- 평균 **14.7회** 이송 거절
- 신고 → 응급실 도착·최초 처치까지 **1시간 32분(92분)**
- **34명 중 13명 사망**, 그중 3명이 10대 미만
- 응급실 1000명당 사망률: 2023년 2~7월 5.7명 → 2024년 같은 기간 **6.6명**

### 🔴 통계 공백

**소방청은 '응급실 뺑뺑이 건수'를 공식 집계·공개하지 않는다** — "오해의 소지가 다분하다"는 이유. 즉 **"뺑뺑이 몇 건"이라는 단일 공식 수치는 존재하지 않으며**, 언론이 인용하는 숫자는 전부 재이송·수용곤란 고지 등 **대리 지표**다.

## 5. 🎯 게임에 바로 쓸 수치 범위

> ⚠️ **이 범위는 「접촉·거절당한 병원 수」 기준**(헤드라인 슬롯 값)이다. 2막 전원 시퀀스의 병원 수(6곳)는 **「PCI 가능 후보 기관 수」 기준**이라 이 범위와 **직접 비교하지 않는다** → [stemi-factsheet.md](stemi-factsheet.md) §2. 두 기준을 섞으면 같은 숫자가 낙관도 되고 축소도 된다.

**실제 범위**: 거부 **8~15곳** · 지연 **1시간 20분~5시간**(분만 3시간 30분)

> **12곳 / 2시간이 가장 '평범한' 수치다.**

⚠️ **이 범위를 벗어나면(예: "50곳 거부", "20시간") 즉시 비현실적으로 읽힌다.** 과장이 리얼리즘을 죽인다.

## 7. 체크리스트 — 헤드라인 하나 쓸 때마다

- [ ] '뺑뺑이'에 작은따옴표 있나
- [ ] 말줄임표가 U+2026 한 문자인가 (`...` 아님)
- [ ] "죽" 안 썼나 → "숨졌다 / 사망"만
- [ ] 환자에게 이름 없나 → 나이+속성만
- [ ] 거부 병원 수가 8~15 범위인가
- [ ] 지연이 1시간 20분~5시간 범위인가
- [ ] 지역·연도·환자 프로필 중 **최소 두 개**가 실제 사건과 어긋나나
- [ ] 실제 병원명·실명 없나
- [ ] 2회차 이상이면 "또" 붙었나
- [ ] 게임이 이 기사에 해석을 덧붙이지 않았나 (숫자가 스스로 말하게)

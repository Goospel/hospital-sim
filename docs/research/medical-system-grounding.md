# 리서치 — 게임 전제 팩트체크 (한국 의료 시스템)

> **목적**: 게임 "수화기 너머의 벽"의 전제·메커니즘이 실제 한국 의료의 구조적 문제를 정확히 반영하는지 웹 리서치(출처 필수) + 적대적 검증으로 확인하고, 왜곡·누락을 교정한다.
> **방법**: 5개 도메인(응급실 뺑뺑이 / STEMI 임상 / 필수의료 붕괴 / 지방 공백 / 2024~25 맥락·톤) 각각 리서치 → 출처 뒷받침 여부 + 게임 전제 적대적 검증 → 종합. (2026-07-16)
> **관련**: [stemi-factsheet.md](stemi-factsheet.md) · [../game-concept.md](../concept/game-concept.md) · [../../plan.md](../../claude-docs/plan.md)

## 한 줄 결론

게임의 **방향**("개인이 최선을 다해도 구조 때문에 진다")은 통계가 강하게 확증한다.
그러나 게임이 그 **'구조'를 '병상 0(NO_BED)'으로 형상화**한 것은 **현실의 지배적 병목을 뒤집는다** — 실제 뺑뺑이 1위 사유는 **전문의/배후진료 부재(41.9%)**이고 병상 부족은 15.0%로 3위다. 이 교정이 이 리서치의 핵심 산출물이다.

---

## ✅ 확증된 것 (게임 방향이 맞다)

- **개인 설득 밖의 구조적 미수용이 지배적** — 재이송 2023년 4,227건 → 2024년 5,657건, 수용곤란 고지 58,520 → 110,033건(+88%). 국회입법조사처: "병상·인력 등 근본 원인이 개선되지 않으면 전공의가 복귀해도 재이송은 해결되지 않는다."
- **"NO_BED=0이면 어떤 설득에도 안 뚫린다"는 하드락 전제는 현실적** — 입법조사처가 "근본 수용불가 원인 미개선 시 개정 지침도 현장에서 작동 못 한다"고 직접 지적.
- **"전화를 돌려 수용처를 찾는" 코어 루프는 임상적으로 정확** — STEMI는 심도자실(cath lab) + 24시간 중재 순환기내과를 갖춘 **PCI 가능 병원**이라야 재관류가 되므로, 아무 병원이나 못 받는다.
- **결정론 코드=판정 / LLM=대사 분리**는 "구조가 결과를 결정한다"는 논지와 구조적으로 정합적.
- **지방 의료 공백(축③)은 실증됨** — 필수의료 전문의 인구 1,000명당 수도권 1.86명 vs 비수도권 0.46명(약 4배). 지방 STEMI 30일 사망률이 대도시의 약 3배(동일 치료 시에도 1.47배).

## ⚠️ 교정해야 하는 것 (안 고치면 의도에 타격)

### 1. [핵심] '구조 = 병상 0' 형상화 → 지배 병목은 '배후진료 불가'
- 2023년 재이송 4,227건 중 **전문의 부재 1,771건(41.9%)** > 기타 26.5% > **병상 부족 635건(15.0%)** > 1차 응급처치 11.3%.
- 병상 부족 635건조차 **71.5%가 '응급실 자리 없음'(과밀)**이고 **수술실 부족은 단 4건**뿐 — '병상 중심 서사'를 오히려 반박한다.
- 진짜 병목은 **배후진료**(응급 초기진료 이후 최종치료: 각 과 전문진료·응급수술·입원·중환자실). 병상·응급의학과 당직이 있어도, 그 환자를 살릴 과(STEMI=순환기 중재시술팀)가 없으면 못 받는다.
- **교정**: 거절 사유를 2종 → **최소 4종**으로. 지배 사유를 '병상 0'이 아니라 **'배후진료 불가(NO_BACKUP_CARE)'**로. STEMI 맥락에선 **'순환기 중재시술팀·카테터실 점유/미가동'**으로 구체화.

### 2. '180초 골든타임'은 실제의 약 30~40배 압축
- 실제: door-to-balloon **90분**, 전원 FMC-to-device **120분**, 이상적 재관류 2시간(최대 12시간). '초 단위 골든타임'은 오정보 소지.
- **교정**: 인트로/결말에 **각색 고지 한 줄** + 실제 기준 병기. (각색 명시가 오히려 리얼리티·방어력↑)

### 3. '시간초과 = 즉사' 이분법
- 실제 예후는 절벽이 아니라 **지연에 비례하는 연속 경사**(폐색 2~3시간 괴사 시작, 24시간 전층 괴사).
- **교정(선택)**: 타이머 구간별 결과 등급(경색 크기·합병증·사망확률) 차등.

### 4. STEMI 정합성 오류
- 리서치가 근거로 든 대표 비극 2건(대구 10대 추락=**신경외과** 부재, 대구 쌍둥이 임신부=**산부인과** 부재)은 **STEMI가 아니다**. STEMI 배후 병목은 '순환기 중재시술팀·카테터실'.
- 저수가 직결 서사도 STEMI엔 비약 — **심장내과 원가보전율은 117%**(저수가 최악은 산부인과 61%·소청과 79%). STEMI 당직 공백의 실제 주드라이버는 저수가보다 **전공의 이탈·번아웃·지방 PCI 병원 부족**.
- **교정**: STEMI 사례·거절 로직을 STEMI에 맞게 교체. 저수가 서사는 (본선 확장 시) 산부인과·소청과 시나리오에 배치.

### 5. 지방 공백의 진짜 킬러 = 지리적 부재
- 축③의 핵심은 '수용 거부'만큼이나 **'골든타임 반경 안에 받을 병원이 애초에 없다'** — 지역심뇌혈관센터 미설치 8개 시·도, 응급의료취약지 98곳, 평균 병원도착 141분.
- **교정(선택)**: 지방 시나리오에 **거리·이송시간 축**(NO_REACHABLE_PCI 또는 이송 페널티) 추가, 후보 병원 수 축소·일부 60분 밖 잠금.

## 🚫 현실엔 지배적인데 게임이 빠뜨린 것 (요약)

- **배후진료 불가** (위 1번) — 필수 추가.
- **지리적 거리/이송시간** (위 5번).
- **응급실 과밀·boarding** — 내원 경증환자 비중 약 44%가 과밀에 기여. 게임은 모든 거절을 공급측 자원 부족으로만 귀결.
- **형사처벌 우려·방어진료** — 이대목동 사건 대법원 **최종 무죄**(2022.12.15)에도 그 사이 소청과 충원율 101%(2018)→16.3%(2023) 붕괴. "기소·수사 과정 자체가 형벌"이라는 수용 위축 심리.
- **혈전용해 교량 치료** — 2시간 내 PCI 불가 시 '혈전용해 후 전원' 대안 존재. 'PCI 아니면 끝'은 실제보다 극단.

## 🎯 톤 가드레일 (특정 집단 비난 금지)

- **'구조' 단일 결말의 위험**: 모든 개별 행위자를 무구하게 만들어 책임을 익명의 구조로 증발시킬 수 있음(특히 2024 의정갈등의 행위성이 구조 뒤로 숨음). → **'구조'를 복수의 구체적 원인 카드**(배후진료·필수의료 인력 붕괴 / 저수가 / 전공의 의존 수련구조 / 지역 불균형 / 2024 급성충격)로 병렬 제시.
- **'거절 = 악' 프레임 방지**: 수용병원 당직의의 정당한 제약(당직 1명이 이미 3명 동시 진료, 시술팀 점유)을 거절 대사에 실어 도덕 문제가 아닌 구조적 딜레마로.
- **실명 금지**: 실존 사망사건·병원·유가족 실명/상징 미사용 → 가상 병원·환자로 재구성(2차가해·지역 낙인 차단).
- **정파 중립**: 특정 장관·단체·정파 미지목. 책임 여론이 다면적(반반)인 상황에 어느 쪽 면죄부도 금지.
- **검증 안 된 수치 배제**: 게임 UI/엔딩에는 출처확인된 통계만, 연도·출처 화면 표기. (검증에서 WEAK/UNSOURCED로 강등된 수치 목록은 아래.)

### 게임에 쓰지 말 것 (검증 실패 수치)
> 1차 출처가 뒷받침 못 하거나 원문에 없어 강등됨 — UI/엔딩에 사실로 노출 금지.
- "15분 단축당 1,000명당 6.3명 사망 감소" · "병상가동률 90% boarding" · "소청과 수입 10년 28% 감소" · "연 700명 입건·40명 기소·20명 유죄" · "흉부외과 정상 수련병원 7.4%" · "빅5 비수도권 환자 71만→79만" · 여론조사 "JTBC 59/34".

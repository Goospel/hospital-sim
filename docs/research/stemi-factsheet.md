## 2. 각색 고지 (게임값 vs 실제값)

| 게임값 | 실제값 | 왜 각색했나 |
|---|---|---|
| **골든타임 180초 타이머** | door-to-balloon 90분 / 전원 FMC-to-device 120분 / 이상적 2시간 | 실시간 텍스트 협상의 긴장을 위해 90~120분 창을 약 30~40배 압축. 초 단위 '골든타임'은 연출 — 인트로에 실제 기준 병기. |
| **타이머 종료 = 즉사(이분법)** | 지연에 비례하는 연속 경사(2~3h 괴사 시작, 24h 전층 괴사) | 명확한 실패 조건·극적 종결을 위한 단순화. 구간별 결과 등급으로 완화 가능. |
| **거절 사유 4종(NO_BED / NO_ER_ONCALL / ER_OVERCROWDED / NO_BACKUP_CARE)** | 전문의 부재 41.9% > 병상 15.0% 등 복합, 지배 병목은 배후진료 불가. 심도자실 점유·형사우려·경증 과밀·거리 부재도 실재 | 규칙을 4종으로 재설계해 **지배 병목=배후진료 불가**를 반영(PR #11). 거리·형사우려 축은 v2 보류. |
| **전화 돌릴 병원 6곳** | **PCI 가능 후보 기관 수 기준** — 취약지·센터 미설치 지역은 후보 1~2곳이거나 60분 밖 | 복수 병원 협상 루프 성립용 낙관 설정. 지방 시나리오에선 후보 축소·거리 잠금(v2).<br>⚠️ **기준이 다르면 방향이 뒤집힌다** — 「접촉·거절당한 병원 수」 기준(보도 표본)은 6보다 **훨씬 많다** → [er-rejection-news-coverage.md](er-rejection-news-coverage.md) §5. **게임의 6곳은 전자 기준으론 낙관, 후자 기준으론 축소다.** PDF에 쓸 때 어느 기준인지 반드시 밝힐 것. |
| **결말 '병원 장부'(미용·검진 흑자 vs 순환기 채용 0명)** | 필수과 저수가·적자 vs 미용·비급여 고마진은 방향 confirmed(부호만 근거). 대형 대학병원이 심장중재 '채용 0'은 압축 표현 | ₩ 액수는 각색, **부호·대소만 근거 준수**. 근거: [essential-care-economics.md](essential-care-economics.md).<br>⚠️ **특정 금액을 인용하지 않는다** — 순이익은 고정값이 아니라 `ledger.ts`가 플레이 결과(부문 손익 + 이번 주 진료·검사·소송)에서 **산출**한다. 같은 게임이 양심 루트 ≈본전 / 공범 루트 큰 흑자로 갈리고, **그 격차가 논지**다. 초안의 "+287억"은 PR #14 시점 고정 시나리오 값이라 지금 코드에 없다. |
| **1막 — 하루 진료 자리 3 · 주당 콜 35통(7일×5) · 과별 의사 상한 3** | 상급종합 응급실의 실제 콜·병상 규모와 무관 | 스케일 압축. **"하루에 앉힐 수 있는 환자(3) < 그날 걸려오는 콜(5)"**이 딜레마의 뼈대다. 상한은 각색값이 아니라 `MAX_DOCTORS_PER_DEPT = FIXED_BEDS` 파생 — 볼 환자가 없으면 4명째는 의미가 없다. |
| **1막 — 콜당 수가/원가 2줄**("진료비 (병원 책정)" vs "수가 (정부 고시)") | 행위 단위 원가보전율(기본진료·응급·수술·처치·검체) → [fee-schedule-and-subsidies.md](fee-schedule-and-subsidies.md) §2 | **금액은 각색, 비율(원가보전율)만 근거.** 단일 출처는 `receiving.ts`의 `CALL_ECONOMICS`·`WORKUP_ECONOMICS`.<br>🔴 **행위 단위다** — 과 단위 수치(심장내과 117%·응급의학과 103%)를 섞으면 **정반대 부호**가 나온다(T-039). 과 단위 흑자는 입력이 아니라 플레이어가 검사를 붙였을 때 장부에서 **창발**한다. |
| **1막 — 검사 boarding**(오늘 검사를 붙이면 그 자리가 **내일** 빈다) | 실제 boarding은 입원 대기·과밀의 복합 결과이며 하루로 딱 떨어지지 않는다 | "어제의 흑자가 오늘의 자리를 먹는다"를 하루 단위로 압축해 달력에 의미를 부여. |
| **1막 — 순환기 1명 = 주간만 / 2명 = 24시간 당직** | 24시간 중재 순환기내과는 PCI 병원의 실제 요건; 당직 1명이 이미 3명 동시 진료 | 인력 규모를 1~2명으로 압축. **2번째 의사가 사는 건 처리량이 아니라 밤**이라는 실제 구조를 규칙으로. 근거: [medical-system-grounding.md](medical-system-grounding.md). |

## 3. 인게임 표기 문구 (현행)

- **인트로 하단**(시작 화면): *"게임 속 180초는 실제 골든타임(급성심근경색 전원 시 첫 의료진 접촉 후 120분 이내 재관류 권고)을 극적으로 압축한 연출입니다."*
- **결말 각주**: *"등장하는 병원·인물·사건은 모두 허구이며, 특정 개인·집단을 비난하지 않습니다."*
- **원칙**: 게임은 결말에서 주제를 **글로 설명하지 않는다**(show-don't-tell). "필수의료 붕괴입니다" 류 서술·통계 문장을 화면에 넣지 않고, 통계·출처·각색 고지는 이 팩트시트와 제출 PDF에 둔다.
- **결말 데이터(해석 없는 사실)**: ① 전원 기록 영수증 — 전원 시도 통수·거절 사유별 집계·받을 수 있던 곳 N/전체·골든타임 소요. ② 병원 장부 — 부문 손익·순이익·신규 채용(순환기 0). 숫자만 병치, 서술 0.

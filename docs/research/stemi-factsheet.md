## 2. 각색 고지 (게임값 vs 실제값)

| 게임값 | 실제값 | 왜 각색했나 |
|---|---|---|
| **골든타임 180초 타이머** | door-to-balloon 90분 / 전원 FMC-to-device 120분 / 이상적 2시간 | 실시간 텍스트 협상의 긴장을 위해 90~120분 창을 약 30~40배 압축. 초 단위 '골든타임'은 연출 — 인트로에 실제 기준 병기. |
| **타이머 종료 = 즉사(이분법)** | 지연에 비례하는 연속 경사(2~3h 괴사 시작, 24h 전층 괴사) | 명확한 실패 조건·극적 종결을 위한 단순화. 구간별 결과 등급으로 완화 가능. |
| **거절 사유 4종(NO_BED / NO_ER_ONCALL / ER_OVERCROWDED / NO_BACKUP_CARE)** | 전문의 부재 41.9% > 병상 15.0% 등 복합, 지배 병목은 배후진료 불가. 심도자실 점유·형사우려·경증 과밀·거리 부재도 실재 | 규칙을 4종으로 재설계해 **지배 병목=배후진료 불가**를 반영(PR #11). 거리·형사우려 축은 v2 보류. |
| **전화 돌릴 병원 6곳** | 취약지·센터 미설치 지역은 후보 1~2곳이거나 60분 밖 | 복수 병원 협상 루프 성립용 낙관 설정. 지방 시나리오에선 후보 축소·거리 잠금(v2). |
| **결말 '병원 장부'(단일 병원 순이익 +287억 · 순환기 채용 0명)** | 필수과 저수가·적자 vs 미용·비급여 고마진은 방향 confirmed(부호만 근거). 대형 대학병원이 심장중재 '채용 0'은 압축 표현 | ₩ 액수는 각색, **부호·대소만 근거 준수**. 근거: [essential-care-economics.md](essential-care-economics.md). |

## 3. 인게임 표기 문구 (현행)

- **인트로 하단**(시작 화면): *"게임 속 180초는 실제 골든타임(급성심근경색 전원 시 첫 의료진 접촉 후 120분 이내 재관류 권고)을 극적으로 압축한 연출입니다."*
- **결말 각주**: *"등장하는 병원·인물·사건은 모두 허구이며, 특정 개인·집단을 비난하지 않습니다."*
- **원칙**: 게임은 결말에서 주제를 **글로 설명하지 않는다**(show-don't-tell). "필수의료 붕괴입니다" 류 서술·통계 문장을 화면에 넣지 않고, 통계·출처·각색 고지는 이 팩트시트와 제출 PDF에 둔다.
- **결말 데이터(해석 없는 사실)**: ① 전원 기록 영수증 — 전원 시도 통수·거절 사유별 집계·받을 수 있던 곳 N/전체·골든타임 소요. ② 병원 장부 — 부문 손익·순이익·신규 채용(순환기 0). 숫자만 병치, 서술 0.

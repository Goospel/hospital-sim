## 1. 수가 산식 — 무엇에 붙는가

```
의료행위비용 = 행위별 상대가치점수 × 환산지수 × 종별가산율
```

**실례**(2020년 논문): 두부 CT(다-245) 773.87점 → 의원 `773.87 × 85.8원 × 115% = 76,358원`

| 항 | 무엇에 붙나 | **진료과별로 다른가?** |
|---|---|---|
| **상대가치점수** | 개별 의료행위(코드) | **아니다** — 행위 단위 |
| **환산지수** | 요양기관 **유형**(의과 의원/병원, 치과, 한방, 약국) | **아니다** — 기관 범주 |
| **종별가산율** | 요양기관 **종별**(의원/병원/종합/상급종합) | **아니다** — 기관 급수 |
| *(별도 층)* **정책가산** | 행위 + **전문의 과목·지역·시설 요건** | **일부 그렇다** |

⚠️ **"유형별 환산지수"의 *유형*을 진료과로 오독하면 안 된다.** 2008년부터 7개 의약단체가 공단과 개별 협상하는데, 여기서 유형은 의과/치과/한방/약국 같은 **기관 범주**다. **내과 환산지수·외과 환산지수 같은 건 존재하지 않는다.**

⚠️ **게임 UI에 "순환기내과 수가 인상" 같은 표현을 쓰면 사실관계가 깨진다.** 정확한 표현은 "그 과가 주로 쓰는 행위 코드의 상대가치점수 인상" 또는 "그 과 대상 정책가산 신설"이다.

**상대가치 3요소**: 업무량(주시술자의 시간·노력) · 진료비용(임상인력·장비·재료) · 위험도(의료사고 분쟁해결 비용). 의사 업무량·위험도는 **명목상 반영된다** — 문제는 반영 수준이 낮고 유형 간 불균등하다는 것.

**결정 주체**: 상대가치운영기획단(15명: 복지부1·공익2·가입자3·공급자6·학계3) + 행위전문위 심의 → **건강보험정책심의위원회(건정심)** 의결(국민건강보험법 제4조).
**환산지수**: 매년 건보공단 ↔ 의약단체 **협상** → 건정심. 2025년 의원 **94.1원**(+0.5%), 병원 **82.2원**(+1.2%).

## 3. 과별 원가보전율 — 가산 후에도 적자인가

**판정: 세 갈래로 갈라진다. "가산에도 불구하고 적자"는 거짓이다.**

| 진료과 | 원가보전율 | 가산 |
|---|---|---|
| 방사선종양학과 | **252%** | — |
| 안과 | 139% | — |
| **심장내과(=순환기내과)** | **117%** | — |
| 마취통증의학과 | 112% | — |
| **응급의학과** | **103%** | 응급가산 100% |
| **심장혈관흉부외과** | **100%** | **흉부외과 가산 100%** |
| 외과 / 신경외과 | 84% | 외과 가산 30% |
| 소아청소년과 | 79% | (2024년 확대 前) |
| 내과 | 72% | 없음 |
| **산부인과** | **61%** | (2023년 정책수가 前) |
| 정신건강의학과 | **55%** | 없음 |

**(1) 가산 받는 과 → 가산 덕에 "정확히 본전"**
흉부외과 **100%**, 응급의학과 **103%**. 100% 가산을 받고서야 원가 100%다.
> **뒤집어 읽으면: 가산을 빼면 원가의 절반.** 이게 "가산에도 불구하고 적자"보다 정확하고 더 극적이다.

**(2) 가산 없는/약한 필수과 → 여전히 명백한 적자**
산부인과 61%, 내과 72%, 소아과 79%, 외과 84%. **29개 과 중 원가를 보전한 건 6개(약 20%)뿐.**

**(3) 단위를 바꾸면 뒤집힌다** → §2.1

## 5. ⭐ 가산금은 병원에 지급되고, 그 과에 줄 법적 의무가 없다

**게임 장치로 가장 강력한 사실.**

| 사실 | 내용 |
|---|---|
| **지급 대상** | 가산 수가는 **요양기관(병원)에 지급**. 병원이 해당 과에 배분할 **법적 의무 없음**. 복지부 「지원기준」은 고시가 아닌 **행정 권고** |
| **의무 배분 비율** | 가산금 대비 **외과 60% / 흉부외과 30% 이상**만 해당 과에 지원하면 됨 |
| **완화 조항** | **가산금 20억 초과 시 외과 40% / 흉부외과 20%로 완화** |
| **유일한 제재** | "지원 실적이 부진한 수련병원은 다음연도 전공의 정원 **5% 범위 내 감축**" |
| **실제 사용처** | **의료기관 수입 50%**, 전문의 인건비 25%, 의국비 5%, 전공의 인건비 4%, 간호사 인건비 3% |
| **전공의 수련보조수당** | 외과 월 **100만원**, 흉부외과 월 **150만원** (필수) |
| **씨앗** | 2009년 정부 원문: *"수가 인상으로 발생한 수익이 흉부외과 및 외과의 처우개선 등에 활용될 수 있도록 **협조해달라**"* — **강제가 아닌 협조 요청** |
| 2018 국정감사 | 흉부외과 가산금 총 **1,850억 4,100만원**. *"해당 지원금이 병원으로 지급돼 실제 흉부외과 의사들에게는 제대로 지급되고 있는지 **파악조차 안되고 있다**"* |

## 6. ⭐ 재정중립 — 왜 필수과 수가가 안 오르나 (악의가 아니라 산수)

**상대가치 개편은 총점 고정(재정중립)이 전제다. 한 유형을 올리려면 다른 유형을 내려야 한다.**

그래서 개편은 구조적으로 **제로섬 재분배**가 되고, 언론은 이를 **"재정 돌려막기식"**, **"나눠먹기식 땜질 처방"**이라 비판한다. 하향 조정은 해당 과가 수용 불가하므로 **"진료과별 총점 고정"**이라는 방법을 쓸 수밖에 없었다 — **개편 실무에서 진료과가 회계 단위로 실제 등장한다.**

**2차 개편**(2017~2020, 매년 25%씩 4년 반영): 검체 **3,637억** + 영상 **1,363억** = 5,000억 인하 + 건보재정 3,500억 = **8,500억**을 수술·처치·기능검사에 투입, 원가보상률 90% 수준 조정. **개편 후에도 검체 142%·영상 116%로 100% 상회.**

**3차 개편**: 종별가산(연 약 5.2조) 및 내·소·정 입원료 가산(연 3,168억) 정비, 영상·검체 과보상분을 입원·수술로 이전. 비판: "재정 돌려막기 식", 외과계는 최소침습 재료비만 인상하고 고난도 수술행위 고려 없음.

## 8. 비급여 — 왜 규제 밖인가 + 교차보조

**법 구조**: 국민건강보험법 **제41조 제3항** → 「요양급여의 기준에 관한 규칙」 **제9조 + 별표2**가 비급여대상을 **열거**한다.

**비급여 = 건보 급여 대상 아님 → 상대가치점수·환산지수 미적용 → 가격을 의료기관이 자율 결정.**

별표2 열거 예(원문): 미용 목적 성형수술(쌍꺼풀·코성형·유방확대축소·지방흡인·주름살제거) / 일상생활 지장 없는 질환(단순 피로, **주근깨, 무모증, 딸기코, 사마귀, 여드름, 탈모**) / 예방진료(본인 희망 건강검진, 예방접종) / 고급병상 추가비용, 보청기·안경, 치과 임플란트

### ⭐ 교차보조 (cross-subsidy)

논문 직접 서술: 의료계는 *"**급여 행위로 인한 손실을 비급여 행위를 고가로 시행하여 보존**"*해왔다. 문재인케어는 비급여를 급여화하면서 동시에 저평가 급여 행위 수가를 인상하는 구조로 설계됐다 — **교차보조의 존재를 정책이 전제하고 있었다.**

> 📌 **게임의 미용 +70억 / 순환기 −12억 구조가 이미 이걸 모델링하고 있다.** 우연이 아니라 현실 구조와 일치한다.

**규모(2023년, 공단 2025-01 발표)**: 총진료비 **133.0조**, 보험자부담금 86.3조, **비급여 20.2조**. 건강보험 **보장률 64.9%**(전년비 −0.8%p). 기관별: 상급종합 70.8% / 종합병원 66.1% / **병원 50.2%** / 의원 57.3%. **2024년 보장률도 64.9%**.

**비급여 구성(2023년 하반기 보고자료, 병원급 4,221억)**: **정형외과 1,170억(27.7%)** 최대 / 도수치료 494억(11.7%) / 1인실 상급병실료 451억(10.7%) / 척추 MRI 187억(4.4%). 상위 10개 항목이 45.3%.

⚠️ **"미용·피부는 아무 규제가 없다"는 부정확하다.** **가격**은 자율이지만 **범위**는 자율이 아니다. 비급여대상은 별표2에 **열거**돼 있고 **임의비급여는 원칙적으로 금지**된다. 의료법 제45조에 따라 **고지·설명 의무**가 있고 비급여 보고제도로 정부에 보고된다. → **"가격 규제 없음, 범위·공시 규제 있음"**

## 9. 응급의료기금

**법적 근거**: 「응급의료에 관한 법률」 **제19조**. **1995년도 설치**. 2003-03부터 심평원 → **복지부**로 관리운영권 이관.

**재원(제20조제1항)**: ① 건보 업무정지 갈음 **과징금** 중 지원액(시행령 제71조: 재난적의료비 65% / **응급의료기금 35%**) ② 출연금·기부금 ③ **정부 출연금** ④ 운용 수익금

**제20조제2항**: 정부는 「도로교통법」 **과태료·범칙금 예상수입액의 20%**를 매 회계연도 세출예산에 계상해야 한다.

> **판정: "과태료가 재원"은 실질적으로 사실이나 경로는 간접적이다.** 과태료가 기금에 직접 귀속되지 않는다 — 전액 일반회계로 들어간 뒤 예상수입액의 20%를 **일반회계전입금**으로 전입한다. 법적으로는 **"과태료 연동 정부출연금"**이지 목적세식 귀속이 아니다.
>
> **정량적으로는 결정적**: 2025년 일반회계전입금 **2,954억** vs 사업비 **2,973억** → **약 99.4%**. 순수 자체수입(203억)은 사업비의 7%.

**일몰**: 부칙 — **제20조제2항 제1호(과태료)**만 **2027-12-31**까지 효력. **범칙금(2호)은 일몰 없음.** 개정 이력(2012·2016·2022)상 **3차례 연장**.

**응급실 적자 총액 ÷ 환자 수 = 1인당 손실**은 **경제학적으로 틀린 조작**이다. 응급실 적자는 대부분 **고정비(24시간 인력·장비 대기)**에서 나온다. 환자를 더 받을수록 고정비가 분산돼 **병상당 손실은 오히려 줄어든다.**

> **정확한 명제**: "환자를 받을수록 적자"가 아니라 **"안 받아도 적자, 받아도 원가에 못 미침"** — 한계손실이 아니라 **구조적 저보상**.

## 11. ⚠️ 2024~2025 데이터 오염 경고

2024-02 의대 증원 발표에 반발해 전공의 약 7,000명이 집단사직했고, 상급종병 진료량·수익 구조가 급변했다.

- **2024년 상급종합병원 47곳 의료이익 −1조 850억** (2023년 −310억 대비 **35배**). 입원환자 1,428만 → 1,094만명(**−23.4%**)
- 🚨 **이 1조 적자를 "저수가 때문"으로 인용하면 인과 오류다.** 직접 원인은 **진료량 붕괴**다. 2023년 적자는 310억에 불과했다.
- 건보 흑자: 2023년 4조 1,276억 → 2024년 1조 7,244억(**−58%**). 재정 전망(NABO): 적자 전환 2026년, 준비금 소진 **2031년 → 2029년**, 10년 누적적자 **+27.8조**
- 비상진료 투입: 2024년 1조 5,146억 + 2025년 1조 6,292억(수가인상분 기준, NABO) / 2024년 총 2조 9,874억(수련병원 지원 포함) — **모순이 아니라 집계 범위 차이. 인용 시 범위 병기 필수.**

> **게임의 "정상 상태" 밸런싱엔 2023 회계연도 이전 데이터를 쓴다.** 2024~25를 쓰면 반드시 특수국면임을 표시할 것. (복지부 2026 상시조정 근거가 **2023 회계연도**인 것은 이 맥락에서 다행이다.)
